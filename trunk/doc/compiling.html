Lucane is build with ant. It needs js.jar and bsf.jar to compile plugins.

Tested versions

  • ant : Ant 1.6.1
  • js.jar : Rhino 1.5R3
  • bsf.jar : Apache BSF 2.3.0

(more recent should also work)

Installing ant

  • Untar ant. It should create a 'apache-ant-1.6.1' directory.
    • We will call this directory $ANT_HOME
  • Unzip Rhino. It should create a 'rhino1_5R3' directory, with a js.jar file in it.
    • Copy this js.jar into $ANT_HOME/lib
  • Untar bsf. It should create a 'bsf-2.3.0' directory.
    • Copy lib/bsf.jar to $ANT_HOME/lib

Compiling Lucane

  • If $ANT_HOME is in your path, simply type 'ant' in Lucane directory.
  • It will (hopefully) generate a lucane.tgz file in a 'build' directory
