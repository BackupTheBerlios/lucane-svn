Lucane est compilé avec maven.

Versions testées

  • maven 1.0rc1

(les versions plus récentes devraient aussi marcher)

Installer maven

  • Décompresser maven. Un répertoire 'maven-1.0-rc1' devrait etre crée.
    • Nous allons appeler ce répertoire $MAVEN_HOME

C'est tout !

Compiler Lucane

  • Si $MAVEN_HOME/bin est dans votre path, tapez simplement 'maven' dans le répertoire des sources.
  • Un répertoire 'dist' contenant server, client et proxy devrait (normalement) être crée

Les "goals" maven disponibles sont :

  • clean-lucane : supprimer les classe générées
  • build-lucane : compiler lucane
  • mkdist-lucane (par defaut) : génerer le répertoire dist
  • mkwinsetup-lucane : créer un installeur windows
  • site-lucane : créer un site avec différents rapports
