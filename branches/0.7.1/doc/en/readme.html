Lucane - README

What is Lucane ?

Lucane is a free groupware platform written in Java, designed with extensibilty in mind.

Some applications are bundled with the platform :

  • A peer to peer quick message service
  • Client/Server and peer to peer file sharing
  • A multi-user chat
  • Forum & Forum administration
  • Shared calendar
  • Personal notes
  • Easy adminisitration
  • SQL Navigator
  • ...

The platform as a whole provides an ease of development for your networked applications :

  • Object based network protocol
  • Client/Server and peer to peer
  • User and groups management with an abstract Store system
  • Multi Language system
  • ...

As an example, the QuickMessage application is made of less than 200 lines of code.

What is a groupware ?

A groupware is a program that allows several people to work together in a simpler and more productive way than only using email. A groupware is strongly network oriented and provides some tools making it possible to distant people to communicate in a synchronous or asynchronous way. For example, it will be possible for users to share files, or even to store them on a server and to work with them at the same time. A lot of tools can be integrated in a groupware. That's why it is so important to remain extensible.
